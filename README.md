## CocoaPods
CocoaPods is the recommended way to add BMPicker to your project.
Add a pod entry for BMScan to your Podfile.

```Ruby
pod 'BMPicker', '~> 0.0.1'
```		
Second, install BMPicker into your project:

```Ruby
pod install
```
